# Privacy

## 로컬 처리

사용자가 선택한 ZIP은 브라우저의 `File`과 Blob range reader로 해석됩니다. 활동·건강·GPS 원본을 애플리케이션 서버, 데이터베이스, 분석 서비스로 업로드하지 않습니다. 새로고침하거나 `새 ZIP`을 누르면 메모리 상태가 사라지며 앱 자체에는 계정이나 영구 저장소가 없습니다.

사진과 영상은 확장자에서 제외하고 압축 해제하지 않습니다. ZIP entry의 절대경로, 상위 디렉터리 이동, 드라이브 경로, NUL 문자, 비정상 크기·압축률을 차단합니다. 한 소스가 손상돼도 다른 정상 ZIP은 계속 처리합니다. 원본 ZIP은 읽기 전용으로 열며 수정하지 않습니다.

## 지도 네트워크

OpenFreeMap 스타일과 지도 타일만 외부에서 받습니다. 경로 좌표는 deck.gl 클라이언트 레이어에만 전달하고 타일 URL이나 요청 본문에 넣지 않습니다. 타일 공급자는 일반적인 네트워크 정보와 사용자가 보고 있는 지도 영역을 알 수 있습니다. 연결 실패 또는 8초 timeout 시 외부 타일이 없는 로컬 배경으로 전환합니다.

## 민감 위치

세 번 이상 반복된 경로 시작·종료 지점이 250m 안에 모이면 privacy 후보로 봅니다. 후보 중심 반경 300m 안의 점은 지도용 LOD와 Poster 입력에서 제거합니다. 이 휴리스틱은 집을 확정하지 않으며 중립적인 `민감 위치 후보`로만 부릅니다.

마스킹으로 두 점 미만이 남는 경로는 안전하게 지도에서 제외합니다. 실제 데이터가 부족해 후보가 검출되지 않을 수도 있으므로 공유 전 사용자가 결과를 확인해야 합니다.

## Poster

Poster는 외부 지도 타일 없이 오프스크린 Canvas에 렌더링합니다. `privacyMasked: true`가 필수이며 UI에서 끌 수 없습니다. `canvas.toBlob("image/png")` 결과에는 앱이 GPS/EXIF 메타데이터를 추가하지 않습니다. 16:9 3840×2160, 4:5 2160×2700, 9:16 2160×3840을 지원합니다.

## 삭제와 재가져오기

- `새 ZIP`: 현재 React 상태, 선택 경로, 지도 포커스를 초기화합니다.
- 새로고침/탭 닫기: 브라우저 메모리의 정규화 데이터가 사라집니다.
- 다른 기간으로 다시 분석: 새 ZIP 화면에서 기간 지정을 선택해 다시 읽습니다.

현재 단계에는 로그인, 클라우드 동기화, 서버 백업이 없습니다.
