# Forecast method

Body Weather Forecast는 의료 예측 모델이 아니라 사용자의 최신 관측을 개인 이력 안에서 설명하는 결정적 요약입니다.

## 기준일과 미래 누출 방지

1. 요청 날짜가 있으면 그 날짜, 없으면 활동과 웰니스 중 가장 최신 날짜를 `asOfDate`로 사용합니다.
2. 기준선은 `asOfDate` 직전 최대 28일만 사용합니다. 기준일 이후 기록은 어떤 factor에도 포함하지 않습니다.
3. 서울 날짜와 기준일 차이가 1일 이하면 `오늘/내일`, 그보다 오래되면 `마지막 관측/다음 날 시뮬레이션`으로 표시합니다.

## 신호와 기본 가중치

| 신호 | 기본 가중치 | 방법 |
| --- | ---: | --- |
| 수면 점수 | 24% | 원본 점수. 없고 수면 시간만 있으면 7.75시간 중심 추정 |
| HRV | 20% | 이전 28일 median/MAD robust score, 높을수록 회복 신호 |
| Body Battery | 20% | 일일 최고 원본 점수 |
| 최근 부하 | 16% | 최근 3일과 이전 26일의 기대 부하 비교 |
| 안정시 심박 | 10% | median/MAD robust score, 낮을수록 회복 신호 |
| 스트레스 | 10% | `100 - 평균 스트레스` |

신호가 없으면 0점을 넣지 않고 남아 있는 factor에 가중치를 비례 재분배합니다. HRV와 안정시 심박은 MAD에 1.4826을 곱해 robust z-score를 만들고 0~100으로 제한합니다. 기준선이 7개 미만이면 원본 범위의 제한적 점수를 사용하거나 상태를 `estimated`로 낮춥니다.

운동 부하는 Training Load, Relative Effort, 이동 시간 기반 대체값 순서로 사용합니다. 대체값은 `estimated`이며 UI 근거에 나타납니다.

## 상태, 권장 밴드, 신뢰도

- 85~100: 맑음 / 강하게
- 65~84: 구름 조금 / 보통
- 45~64: 흐림 / 가볍게
- 25~44: 비 / 회복 우선
- 0~24: 폭풍 / 회복 우선
- 계산 신호 없음: 관측 부족 / 판단 보류

신뢰도 `high`는 건강 신호 3개 이상, 기준선 14일 이상, 최신성 1일 이하입니다. `medium`은 건강 신호 2개 이상, 기준선 7일 이상, 최신성 3일 이하이며 나머지는 `low`입니다.

## 다음 날 시뮬레이션

오늘 계획의 가정은 휴식 +8, 가볍게 +4, 보통 0, 고강도 -9를 현재 점수에 더합니다. 신뢰도별 불확실성은 ±5, ±9, ±15입니다. 이는 실제 미래 데이터를 사용하지 않는 what-if 범위이며 확정 예측이 아닙니다.

단위 테스트는 동일 입력 결정성, 오래된 데이터 라벨, Tier 강등, 신호 없음, 시나리오 변화와 미래 데이터 누출 방지를 검증합니다.
